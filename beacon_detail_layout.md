# Beacon Detail Layout

## 1. Header Section
*   **Primary Title**: Beacon Name (e.g., "PRESIDIO TUNNEL TOPS")
    *   *Style*: High-contrast Serif, Uppercase, pure white.
*   **Subtitle**: Nickname / Vibe (e.g., "The Sunday Reset")
    *   *Style*: Modern Sans-Serif, Bold, Cyan/Electric Blue glow.

## 2. Hero Composition ("Sticker" Layer)
A layered assembly of irregular die-cut shapes.
*   **Layer A (Back)**: Beacon Image
    *   Isometric view of the location.
    *   Thick white die-cut border.
*   **Layer B (Front)**: Crowd/Memory Cluster
    *   Cluster of user avatars/characters.
    *   Thick white die-cut border.
    *   *Position*: Overlaps bottom-left of the Beacon Image.

## 3. Metadata Section
Located adjacent to or slightly underneath the sticker composition.
*   **Tags**: Crowd Labels (e.g., "Families", "Wellness Crowd")
    *   Pill-shaped containers, translucent dark fill with cyan border.

## 4. Narrative Footer
*   **Description**: Short summary text.
    *   *Style*: Readable Sans-Serif, slightly muted color (Off-white/Grey).
